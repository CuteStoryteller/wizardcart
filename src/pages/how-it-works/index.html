The primary goal of WizardCart is to automatically fill product cards. It manages the entire process:
  • analyzing which product cards need to be filled on your website
  • downloading images and description from supplier websites
  • optimizing images
  • navigating through OpenCart
  • filling product cards with downloaded images and description

Simply queue the desired task. You have two options: fill an individual product or update all products of a specific brand.
  • For an individual product, provide either product ID or product page URL from your website
  • For a brand-wide update, specify a brand name, and WizardCart will fill in product cards that are missing a main image or description
Once processing starts, tasks will be handled one by one. You can click stop processing button at any time but the process will not stop immediately. Make sure to check your internet connection, as WizardCart may not function properly with a slow one.

While tasks are being processed, you will receive real-time status updates. Once a brand task is completed (whether successfully or not), you can copy to clipboard all details about brand products that need to be filled in. You can also redo a task by clicking the corresponding button, or delete it.
For task processing the following files are required:
  • admin.json: OpenCart data of your website to fill product cards
  • brands.json: brand information
  • scraper.json: HTML structure details of your website and supplier (partner) websites for extracting product data

To configure scraper.json refer to the product-scraper library documentation. For admin.json, check the cartmin library documentation. Additionally admin.json must specify the maximum size and dimensions of images allowed by OpenCart. Each brand entry in brands.json must include the supplier name and the corresponding OpenCart image directory path. Try to keep the directory structure in OpenCart's file manager as simple as possible. Finally, each partner's scraper-like file need to be specified.
Click on a file to open it in the editor. Remember that while tasks are being processed, you cannot modify settings or config files. Additionally, you cannot delete, rename or edit admin.json, brands.json, scraper.json at any time.

What you can do:
  • add a new file (partner scraper) by clicking button "add a new file" or pressing "Ctrl + n"
  • save a file by clicking the save button or pressing "Ctrl + s". If the file cannot be parsed as JSON, the save button will turn red, and corrupted version will not be saved
  • delete a file by selecting it and pressing "Delete"
  • rename a file by selecting it and pressing "r"
